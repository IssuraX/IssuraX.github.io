리뉴얼 반응형 SCSS 구조화 템플릿화

요약

자사 홈페이지 리뉴얼 과정에서 반응형 UI를 구축하고, SCSS 구조화와 공통 템플릿을 도입해 유지보수성과 제작 효율을 개선했습니다.

문제 · 과제

  • 기존 정적 구조로 모바일 가독성이 낮고 수정 비용이 큼
  • 콘텐츠 확장 시 매번 레이아웃을 다시 손봐야 함

해결

  • SCSS 변수/믹스인/컴포넌트 단위로 스타일 구조를 재정리
  • 공통 레이아웃/섹션 템플릿을 만들어 신규 페이지 확장 가능 구조로 개선

결과

  • 신규 페이지 제작 시간 약 n% 단축(수치 교체)
  • 모바일 UI 안정성 및 유지보수 효율 향상
면접에서 강조 포인트: 리뉴얼 시 ‘확장성’을 기준으로 템플릿/컴포넌트를 설계한 경험

프로젝트 정보

  • 기간 2022.12–2023.01
  • 소속 위킵(주)
  • 역할 퍼블리싱 전담
  • 기여도 100%
  • 스택 HTML/CSS(SCSS), JavaScript/jQuery, Git, (Jira/Slack)
