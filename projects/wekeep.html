Overview

프로젝트 요약

자사 홈페이지 리뉴얼 과정에서 반응형 UI를 구축하고, SCSS 구조화와 공통 템플릿을 도입해 유지보수성과 제작 효율을 개선했습니다.

단순 리뉴얼에 그치지 않고 이후 콘텐츠 추가·변경 시 빠르게 대응할 수 있는 확장 가능한 퍼블리싱 구조를 구축하는 데 초점을 뒀습니다.

Problem

문제 · 과제

  • 기존 정적 구조로 모바일 가독성이 낮고, 반응형 대응이 부족하여 사용자 경험 저하
  • 스타일 구조가 분산되어 있어 콘텐츠 확장 시 매번 레이아웃을 다시 손봐야 하는 비효율 발생

Solution

해결 방법

  • SCSS 변수/믹스인/컴포넌트 단위로 스타일 구조를 재정리하여 일관성 있는 디자인 시스템 구축
  • 공통 레이아웃·섹션 템플릿을 정의해 신규 페이지 추가 시 기본 구조를 그대로 활용 가능한 확장 구조로 개선
  • Flex/Grid 기반 반응형 레이아웃 적용으로 모든 뷰포트에서 안정적인 UI 제공

Result

결과

  • 신규 페이지 제작 시간 약 30% 단축 — 공통 템플릿 활용으로 반복 작업 감소
  • 모바일 UI 안정성 개선 및 반응형 대응으로 사용자 접근성 향상
  • 유지보수 효율 향상 — 스타일 구조 명확화로 수정 범위 예측 가능
